# Setup 
1. create a virtual environment
```
python -m venv .venv
```
2. Activate the environment
for Linux and macOS
```
source .venv/bin/activate
```
or for Windows
```
.venv\Scripts\activate.bat
```
3. Install the necessary libraries
```
pip install -r requirements.txt
```

4. Run the project using 
```
python gui.py
```
